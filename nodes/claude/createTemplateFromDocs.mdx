---
title: "Create Template From Document"
description: "Generate a structured template based on a provided document"
---

## Configuration

- **Node Type:** action

### Input Fields

| Field Name  | Data Type | Required | Description                             |
| ----------- | --------- | -------- | --------------------------------------- |
| User Prompt | Textarea  | True     | The prompt specifying the document name |
| Context     | Text      | True     | The context for Document                |

```json
{
  "userPrompt": "<string>",
  "userContext": "<string>"
}
```

### Output Fields

| Field Name | Data Type | Required | Description                              |
| ---------- | --------- | -------- | ---------------------------------------- |
|            | String    | True     | A template from document generated by AI |

```json
"<string>"
```
