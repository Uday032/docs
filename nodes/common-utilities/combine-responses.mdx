---
title: "Combine Responses"
description: "Combine two responses into one"
---

## Configuration

- **Node Type:** action

### Input Fields

| Field Name      | Data Type | Required | Description     |
| --------------- | --------- | -------- | --------------- |
| Common Variable | Text      | True     | Common variable |
| Response 1      | Array     | True     | First response  |
| Response 2      | Array     | True     | Second response |

```json
{
  "commonVariable": "phoneNumber",
  "response1": [
    { "phoneNumber": "123456789", "name": "John Doe" },
    { "phoneNumber": "987654321", "name": "Mark Doe" }
  ],
  "response2": [
    { "phoneNumber": "123456789", "position": "Manager" },
    { "phoneNumber": "2378264298", "position": "Director" }
  ]
}
```

### Output Fields

| Field Name | Data Type | Required | Description       |
| ---------- | --------- | -------- | ----------------- |
|            | Array     | True     | Combined response |

```json
[
  { "phoneNumber": "123456789", "name": "John Doe", "position": "Manager" },
  { "phoneNumber": "987654321", "name": "Mark Doe" },
  { "phoneNumber": "2378264298", "position": "Director" }
]
```
