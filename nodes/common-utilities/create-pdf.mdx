# Create PDF

The Create PDF node enables you to generate PDF documents from various input formats including HTML, text, and markdown.

## Overview

This node provides functionality to:

- Convert HTML to PDF
- Generate PDFs from markdown
- Create PDFs from text content
- Add headers and footers
- Include page numbers

## Configuration

| Parameter | Type   | Description                        |
| --------- | ------ | ---------------------------------- |
| Content   | String | Input content (HTML/Markdown/Text) |
| Format    | String | Input format type                  |
| Options   | Object | PDF generation options             |
| Metadata  | Object | PDF metadata (title, author, etc.) |

## Output

The node generates:

- PDF file output
- Success/failure status
- File metadata
- Error messages if generation fails

## Example Usage

```json
{
  "content": "<h1>Hello World</h1><p>This is a test document</p>",
  "format": "html",
  "options": {
    "pageSize": "A4",
    "margin": "1cm",
    "addPageNumbers": true
  },
  "metadata": {
    "title": "Generated Document",
    "author": "System"
  }
}
```
