---
title: "Extract Text from Docx"
description: "Extract text from a DOCX document"
---

## Configuration

- **Node Type:** action

### Input Fields

| Field Name    | Data Type | Required | Description        |
| ------------- | --------- | -------- | ------------------ |
| Document Link | Text      | True     | The DOCX file Link |

```json
{ "documentLink": "<document link>" },
```

### Output Fields

| Field Name     | Data Type | Required | Description    |
| -------------- | --------- | -------- | -------------- |
| Extracted Text | Text      | True     | Extracted text |

```json
{ "text": "<extracted text>" },
```
