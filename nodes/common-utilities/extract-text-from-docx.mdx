# Extract Text from DOCX

The Extract Text from DOCX node allows you to extract text content from Microsoft Word (.docx) files.

## Overview

This node provides functionality to:

- Extract plain text from DOCX files
- Preserve formatting (optional)
- Extract headers and footers
- Handle tables and lists
- Extract metadata

## Configuration

| Parameter       | Type    | Description                         |
| --------------- | ------- | ----------------------------------- |
| Input File      | File    | The DOCX file to process            |
| Preserve Format | Boolean | Whether to preserve text formatting |
| Include Headers | Boolean | Extract headers and footers         |
| Extract Tables  | Boolean | Whether to process tables           |

## Output

The node provides:

- Extracted text content
- Document structure (optional)
- Metadata information
- Success/failure status
- Any error messages

## Example Usage

```json
{
  "input_file": "document.docx",
  "preserve_format": true,
  "include_headers": true,
  "extract_tables": true
}
```
